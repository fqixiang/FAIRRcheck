# ClimSim Analysis Pipeline

**Author:** C. de Wit  
**Version:** 0.3.0  
**Description:** Lightweight climate simulation post-processing pipeline for
analysing output from the ClimSim benchmark dataset. Designed to run on
shared HPC clusters without root access.

This project is openly shared for research purposes.

## Requirements

- Python 3.11+
- numpy, xarray, matplotlib

## Usage

```bash
# Run the main analysis
python main.py --config config/params.yml

# Plot results
python plot.py --input results/summary.nc
```

## Project Structure

```
config/      Analysis configuration files
data/        Input datasets (not included — see Data section below)
results/     Output files generated by main.py
```

## Data

Input data is derived from the ClimSim benchmark (Yu et al., 2023).
Download instructions are in `data/README.txt`.
